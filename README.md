# VecEditor

Учебный проект: векторный графический редактор на C# с архитектурой MVVM.

## Цели
- Рисование и редактирование базовых фигур (line/rect/circle/…)
- Инструменты (выбор/перемещение/масштабирование/удаление)
- Экспорт/импорт (JSON, SVG, PNG)
- Покрытие ключевой логики тестами

## Требования (Build Requirements)
- .NET SDK: **10.0.x**
- Visual Studio 2026 (обновлённая версия) с workload: **.NET Desktop Development**
- ОС: Windows (рекомендуется), Linux/macOS — по возможности

Пакеты (NuGet) ставить вручную не нужно — они подтягиваются командой `restore`.

## Структура репозитория
* `src/VecEditor.App/` — **GUI команда**

  * Avalonia UI: окна, контролы, темы, ресурсы, биндинги
* `src/VecEditor.Core/` — **Geometry команда**

  * Геометрия, фигуры, базовые алгоритмы (без Avalonia)
* `src/VecEditor.ViewModel/` — **ViewModel команда**

  * MVVM: инструменты, команды, состояние редактора, взаимодействие с Core
* `src/VecEditor.IO/` — **IO команда**

  * Сохранение/загрузка проекта (JSON) и экспорт (SVG/PNG)
* `tests/` — **Тестировщики**

  * `VecEditor.Core.Tests` — тесты геометрии/фигур
  * `VecEditor.IO.Tests` — тесты сериализации/экспорта
